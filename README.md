# Rust

## Install

- Install RUSTUP-INIT
    - https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe for 64 Bits
    
    For any other version visit https://www.rust-lang.org/learn/get-started

- After install exec:
    cargo --version


## Making compile

- Create a .rs file
- Write your code
- run: rustc filename.rs
- run: filename
